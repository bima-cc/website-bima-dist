关于我们 ​

北京必码科技工作室（简称：必码），专注于开源项目定制开发与技术服务。我们秉承"只写必要的代码"的核心理念，致力于为客户提供轻量化、贴合业务需求的技术解决方案，让开源技术真正为业务创造价值。

我们的理念 ​

必码坚持"小而精、专而深"的服务原则，聚焦开源生态，专注技术落地。我们深知企业在使用开源产品时面临的"适配难、定制难"痛点，凭借丰富的开源开发经验，从需求分析到交付运维，全程细致跟进，力求让客户以最小的成本解决最实际的问题。

技术能力 ​

我们拥有丰富的开源技术栈和项目经验，主要涵盖以下领域：

  • 身份认证与授权：基于 CAS 的单点登录系统定制开发
  • 微服务架构：Spring Boot、Dubbo、Spring Cloud 等微服务框架集成
  • 中间件集成：Elasticsearch、Kafka、MongoDB、RabbitMQ、Redis、RocketMQ 等
  • AI 能力集成：Ollama 本地模型和 OpenAI API 调用
  • 容器化部署：Docker、Kubernetes 等容器技术应用

业务服务 ​

CAS 单点登录系统 ​

基于 Apereo 基金会维护的 CAS（中央认证服务）项目，提供轻量化定制开发，解决企业多系统单点登录问题。我们基于不同版本的 CAS Overlay 进行了定制化开发，提供了完整的开发基座、配置模板和部署方案。

主要特性：

  • 架构转型：从传统 WAR 包架构升级为现代 JAR 包架构
  • 自定义认证：实现灵活的身份验证逻辑
  • 安全加固：关闭 CAS 默认账号密码登录，提升系统安全性
  • 数据库集成：集成 MyBatis 数据库访问框架
  • 资源提取：全量提取前端静态资源和模板，方便定制
  • 多语言支持：内置中文、英文等多语言支持
  • OAuth 2.0 支持：完整实现标准 OAuth 2.0 认证流程

版本选择：

  • 7.3.x 系列
  • 6.6.x 系列
  • 5.3.x 系列

智能中间件集成验证脚手架 ​

面向开发者打造的一站式微服务项目脚手架，深度融合 AI 能力与主流中间件集成方案，支持开发者快速完成各类常用中间件服务的集成、验证与测试，大幅提升开发效率。

主要特性：

  • AI接口能力：可调用Ollama本地模型和OpenAI API
  • 多中间件集成：集成了Elasticsearch、Kafka、MongoDB、MyBatis、RabbitMQ、Redis、RocketMQ等常用中间件
  • 统一Web界面：基于LayUI框架的直观Web界面
  • 多数据源支持：内置多数据源配置
  • OAuth2.0客户端：集成OAuth2客户端认证机制

版本选择：

  • Spring Boot 版本
  • Dubbo 版本
  • Spring Cloud 版本

关于我们 ​

北京必码科技工作室（简称"必码"），是一家专注于开源项目定制开发与技术服务的工作室。我们秉承"只写必要的代码"的核心理念，围绕企业级身份认证与微服务架构两大核心方向，提供轻量化、可落地的技术解决方案。

我们的理念 ​

必码坚持"小而精、专而深"的服务原则，聚焦开源生态，专注技术落地。我们深知企业在使用开源产品时面临的"适配难、定制难"痛点，凭借丰富的开源二次开发经验，从需求分析到交付运维，全程细致跟进，力求以最小的成本解决最实际的问题。

技术能力 ​

我们深耕 Java 后端技术生态，主要涵盖以下领域：

  • 身份认证与授权：基于 Apereo CAS 的企业级单点登录系统定制开发，覆盖 OAuth 2.0 协议集成、自定义认证处理器、多因素认证等场景
  • 微服务架构：Spring Boot、Dubbo、Spring Cloud 三种架构形态的设计、选型与落地实施
  • 中间件集成：Elasticsearch、Kafka、MongoDB、RabbitMQ、Redis、RocketMQ、MyBatis 等主流中间件的深度集成与调优
  • AI 能力集成：Spring AI + Ollama 本地大模型、OpenAI API 对接、SSE 流式响应、嵌入向量等 AI 应用落地
  • 容器化与部署：Docker 多阶段构建、Jenkins CI/CD 流水线、Kubernetes 容器编排

业务服务 ​

CAS 单点登录系统 ​

基于 Apereo 基金会维护的 CAS（Central Authentication Service）项目，提供面向企业的轻量化定制开发服务，解决多系统统一身份认证问题。覆盖 5.3.x、6.6.x、7.3.x 三个大版本线，支持 Gradle 和 Maven 双构建体系，提供完整的开发基座、配置模板和部署方案。

核心能力： WAR 转 JAR 现代化架构升级 · 自定义认证处理器 · 安全加固 · MyBatis 数据集成 · 前端资源全量提取 · 国际化多语言 · OAuth 2.0 协议集成 · Redis 分布式票据注册表 · 监控与日志体系

查看 CAS 版本文档 · 阅读 CAS 深度解析系列

智能中间件集成验证脚手架 ​

面向开发者打造的一站式微服务项目脚手架，深度融合 AI 能力与主流中间件集成方案。同一业务场景在 Spring Boot 单体、Dubbo RPC、Spring Cloud 微服务三种架构下的完整实现，为架构选型提供可直接对比的参考。

核心能力： 7 大中间件集成验证 · AI 聊天与嵌入向量 · SSE 流式响应 · 多数据源配置 · OAuth 2.0 统一认证 · Docker CI/CD · Maven 多模块工程化

查看脚手架文档 · 阅读 Smart-Scaffold 深度解析系列

CAS SSO 单点登录

基于 Apereo CAS 深度定制的企业级单点登录产品，覆盖 7.3.x / 6.6.x / 5.3.x 三大版本线，支持 Gradle 与 Maven 双构建体系，提供从认证定制到容器化部署的完整能力。

Keycloak SPI 扩展开发

面向开发者的 Keycloak SPI 扩展开发示例，涵盖国密算法、事件监听、用户存储核心功能，提供完整的开发示例和最佳实践，帮助开发者快速构建符合企业需求的 Keycloak 扩展。

Keycloak Sandbox 双沙箱

面向开发者的一站式 Keycloak SPI 开发环境，支持快速开发、测试和部署 Keycloak SPI，提供完整开发基座、多版本支持和内置沙箱环境，帮助开发者快速构建 Keycloak 扩展。

Smart Scaffold 智能脚手架

面向开发者的一站式微服务脚手架产品，提供 Spring Boot / Dubbo / Spring Cloud 三种架构形态，集成 7 大主流中间件并深度融合 AI 能力，为架构选型提供可对比参考。

常用 Linux 命令技术文档 ​

简介 ​

Linux 是一种自由开放源码的操作系统内核，由 Linus Torvalds 于 1991 年首次发布。Linux 系统以其稳定性、安全性、灵活性和强大的命令行界面而闻名，被广泛应用于服务器、嵌入式设备、超级计算机和云计算平台。

Linux 命令行提供了强大而灵活的系统管理能力，掌握常用的 Linux 命令对于系统管理员、开发人员和运维工程师至关重要。Linux 命令通常遵循 POSIX 标准，具有高度的可移植性。

安装方法 ​

Linux 本身是一个操作系统，不能像普通软件一样安装。以下是常见的获取和安装 Linux 系统的方法：

安装完整 Linux 发行版 ​

Ubuntu 桌面版/服务器版 ​

  1. 访问 https://ubuntu.com/download
  2. 下载 ISO 镜像文件
  3. 创建可启动 USB 或 DVD
  4. 重启计算机并从 USB/DVD 启动
  5. 按照安装向导完成安装

CentOS/RHEL ​

  1. 访问 https://www.centos.org/ 或 Red Hat 官网
  2. 下载 ISO 镜像
  3. 按照官方指南安装

语法 ​

Linux 命令的基本语法结构：

command [options] [arguments]
1

常用通用选项：

  • -h, --help: 显示帮助信息
  • -v, --version: 显示版本信息
  • -V, --verbose: 详细输出
  • -q, --quiet: 静默模式

常用命令 ​

文件和目录操作 ​

bash
# 列出目录内容
ls
ls -la                    # 详细信息，包含隐藏文件
ls -lh                    # 人类可读格式
ls -R                     # 递归列出子目录

# 切换目录
cd /home/user             # 绝对路径
cd ..                     # 返回上级目录
cd ~                      # 回到家目录
cd -                      # 切换到上次目录

# 显示当前路径
pwd

# 创建目录
mkdir dirname
mkdir -p dir1/dir2/dir3   # 递归创建目录

# 删除文件和目录
rm filename               # 删除文件
rm -r directory           # 删除目录及内容
rm -rf directory          # 强制删除，无提示

# 复制文件和目录
cp source.txt dest.txt    # 复制文件
cp -r sourcedir destdir   # 复制目录

# 移动或重命名
mv oldname.txt newname.txt # 重命名文件
mv file.txt dir/          # 移动文件到目录

# 查看文件内容
cat file.txt              # 显示文件全部内容
less file.txt             # 分页查看文件
more file.txt             # 分页查看文件（较老的方式）
head file.txt             # 显示文件开头几行
tail file.txt             # 显示文件末尾几行
tail -f file.log          # 实时查看文件变化

# 创建文件
touch filename            # 创建空文件或更新时间戳
echo "content" > file.txt # 写入内容到文件
echo "content" >> file.txt # 追加内容到文件
</command>

### 系统信息查看

```bash
# 系统信息
uname -a                  # 显示所有系统信息
uname -r                  # 内核版本
hostname                  # 主机名
whoami                    # 当前用户

# 硬件信息
lscpu                     # CPU 信息
lsmem                     # 内存信息
lsblk                     # 块设备信息
lspci                     # PCI 设备信息
lsusb                     # USB 设备信息

# 磁盘空间
df -h                     # 文件系统磁盘使用情况
du -sh directory          # 目录大小统计
du -ah directory          # 详细目录大小

# 内存使用
free -h                   # 内存使用情况
cat /proc/meminfo         # 详细内存信息

# 进程管理
ps aux                    # 查看所有进程
ps -ef                    # 查看所有进程（另一种格式）
top                       # 动态查看进程
htop                      # 更好的进程查看器
kill PID                  # 终止进程
kill -9 PID               # 强制终止进程
killall process_name      # 按名称终止进程

# 系统负载
uptime                    # 系统运行时间
w                         # 当前登录用户及系统负载
last                      # 最近登录记录
</command>

命令行效率提升 ​

bash
# 创建别名
alias ll='ls -la'
alias la='ls -A'
alias ..='cd ..'

# 命令历史
history | grep command_name     # 查找历史命令
!!                           # 执行上一条命令
!n                           # 执行历史命令第 n 行
!?string                     # 执行最近包含 string 的命令

# 文件搜索
find /path -name "*.txt"     # 按名称查找
locate filename              # 快速定位文件（需先建立索引）
which command                # 查找命令路径
whereis command              # 查找命令及其文档位置

# 文本处理
grep -rn "pattern" .         # 递归搜索文本
cut -d',' -f1,3 file.csv    # 按分隔符提取列
sort file.txt | uniq         # 排序并去重
wc -l file.txt               # 统计行数

# 系统监控
htop                         # 交互式进程查看
iotop                        # 磁盘 I/O 监控
iftop                        # 网络流量监控
du -sh /path/*              # 目录大小统计

ss -tuln                     # 现代化的 netstat 替代

# 任务调度
crontab -e                   # 编辑定时任务

最佳实践 ​

  1. 使用 Tab 键自动补全：提高输入效率，减少拼写错误
  2. 学会使用 man 命令：man command 查看命令详细手册
  3. 使用管道和重定向：组合命令实现复杂功能
  4. 备份重要数据：定期备份关键文件和配置
  5. 权限最小化原则：避免使用 root 权限执行不必要的操作
  6. 学习正则表达式：提高文本处理效率
  7. 使用别名：简化常用长命令
  8. 熟悉快捷键：如 Ctrl+C 终止、Ctrl+Z 暂停等
  9. 利用命令历史：使用 history 和相关快捷方式
  10. 使用版本控制：对重要的配置文件使用 Git 等版本控制

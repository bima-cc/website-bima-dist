常用 Apache Tomcat 命令技术文档 ​

简介 ​

Apache Tomcat（通常简称为 Tomcat）是 Apache 软件基金会 Jakarta 项目的一个开源实现，是一个免费的、开放源代码的 Web 应用服务器。Tomcat 实现了 Java Servlet、JavaServer Pages（JSP）、Java Expression Language 和 Java WebSocket 技术规范，是一个轻量级的应用服务器，广泛用于开发和部署 Java Web 应用程序。

Tomcat 的核心组件包括：

  • Catalina：Tomcat 的 Servlet 容器实现
  • Coyote：HTTP 连接器组件
  • Jasper：JSP 引擎
  • Naming：JNDI 实现
  • Util：通用工具类

Tomcat 广泛应用于 Java Web 应用开发、企业级应用部署、微服务架构等场景。

安装方法 ​

macOS 系统 ​

使用 Homebrew 安装 ​

bash
# 安装 Java（Tomcat 需要 Java 环境）
brew install openjdk

# 设置 JAVA_HOME
echo 'export JAVA_HOME=$(/usr/libexec/java_home)' >> ~/.zshrc
source ~/.zshrc

# 安装 Tomcat
brew install tomcat

# 验证安装
catalina version

Docker 方式安装 ​

bash
# 运行 Tomcat 容器
docker run -it --rm -p 8080:8080 tomcat:10.1-jdk11

# 运行带应用部署的 Tomcat
docker run -d --name my-tomcat -p 8080:8080 -v /path/to/your/app.war:/usr/local/tomcat/webapps/app.war tomcat:10.1-jdk11

# 使用自定义配置运行
docker run -d --name my-tomcat -p 8080:8080 \
  -v /path/to/server.xml:/usr/local/tomcat/conf/server.xml \
  -v /path/to/context.xml:/usr/local/tomcat/conf/context.xml \
  tomcat:10.1-jdk11

# 使用 docker-compose 运行
cat <<EOF > docker-compose.yml
version: '3.8'
services:
  tomcat:
    image: tomcat:10.1-jdk11
    ports:
      - "8080:8080"
    volumes:
      - ./webapps:/usr/local/tomcat/webapps
      - ./conf:/usr/local/tomcat/conf
    environment:
      - CATALINA_OPTS=-Xmx512m -Xms256m
EOF

# 启动服务
docker-compose up -d

server.xml 配置文件 ​

xml
<?xml version="1.0" encoding="UTF-8"?>
<Server port="8005" shutdown="SHUTDOWN">
  <Listener className="org.apache.catalina.startup.VersionLoggerListener" />
  <Listener className="org.apache.catalina.core.AprLifecycleListener" SSLEngine="on" />
  <Listener className="org.apache.catalina.core.JreMemoryLeakPreventionListener" />
  <Listener className="org.apache.catalina.mbeans.GlobalResourcesLifecycleListener" />
  <Listener className="org.apache.catalina.core.ThreadLocalLeakPreventionListener" />

  <!-- 全局 JNDI 资源 -->
  <GlobalNamingResources>
    <Resource name="UserDatabase" auth="Container"
              type="org.apache.catalina.UserDatabase"
              description="User database that can be updated and saved"
              factory="org.apache.catalina.users.MemoryUserDatabaseFactory"
              pathname="conf/tomcat-users.xml" />
  </GlobalNamingResources>

  <Service name="Catalina">
    <!-- HTTP 连接器 -->
    <Connector port="8080" protocol="HTTP/1.1"
               connectionTimeout="20000"
               redirectPort="8443" />

    <!-- AJP 连接器 -->
    <Connector port="8009" protocol="AJP/1.3" redirectPort="8443" />

    <!-- SSL 连接器示例 -->
    <!--
    <Connector port="8443" protocol="org.apache.coyote.http11.Http11NioProtocol"
               maxThreads="150" SSLEnabled="true">
      <SSLHostConfig>
        <Certificate certificateKeystoreFile="conf/localhost-rsa.jks"
                     type="RSA" />
      </SSLHostConfig>
    </Connector>
    -->

    <Engine name="Catalina" defaultHost="localhost">
      <Realm className="org.apache.catalina.realm.LockOutRealm">
        <Realm className="org.apache.catalina.realm.UserDatabaseRealm"
               resourceName="UserDatabase"/>
      </Realm>

      <Host name="localhost"  appBase="webapps"
            unpackWARs="true" autoDeploy="true">
        <Valve className="org.apache.catalina.valves.AccessLogValve" directory="logs"
               prefix="localhost_access_log" suffix=".txt"
               pattern="%h %l %u %t &quot;%r&quot; %s %b" />
      </Host>
    </Engine>
  </Service>
</Server>

Tomcat 服务管理 ​

启动 Tomcat ​

bash
# 使用 catalina.sh 启动（Linux/macOS）
$CATALINA_HOME/bin/catalina.sh start

# 直接运行 startup.sh
$CATALINA_HOME/bin/startup.sh

# 在后台运行并重定向输出
$CATALINA_HOME/bin/startup.sh > /dev/null 2>&1 &

# 使用 catalina.sh 运行（前台）
$CATALINA_HOME/bin/catalina.sh run

Nacos 常用命令 ​

简介 ​

Nacos（Dynamic Naming and Configuration Service）是阿里巴巴开源的一个动态服务发现、配置管理和服务管理平台。它可以帮助您轻松地构建云原生应用和微服务体系，提供服务发现、配置管理、动态 DNS 服务和服务及元数据管理等功能。

Nacos 致力于帮助开发者发现、配置和管理微服务，从而解决微服务架构中的服务治理问题。

主要特性 ​

  • 服务发现和服务健康监测: 支持多种服务注册中心协议，提供健康检查机制
  • 动态配置服务: 动态配置推送，配置变更实时生效
  • 动态 DNS 服务: 支持权重路由，让服务更稳定
  • 服务及其元数据管理: 管理服务的元数据、流量路由和安全配置

安装 ​

环境要求 ​

  • 操作系统: Linux/Unix/Mac 或 Windows
  • JDK: 64-bit JDK 1.8+
  • Maven: 3.2.x+
  • Git: 最新版

1. 二进制安装（推荐） ​

# 解压
tar -zxvf nacos-server-2.3.0.tar.gz

Parameters:
- serviceName: 服务名

Consumer 服务消费者 ​

java
@SpringBootApplication
@EnableDiscoveryClient
public class ConsumerApplication {
    public static void main(String[] args) {
        SpringApplication.run(ConsumerApplication.class, args);
    }
    
    @Bean
    @LoadBalanced
    public RestTemplate restTemplate() {
        return new RestTemplate();
    }
}

@RestController
@RequestMapping("/consumer")
public class ConsumerController {
    
    @Autowired
    private RestTemplate restTemplate;
    
    @Autowired
    private LoadBalancerClient loadBalancerClient;
    
    @GetMapping("/hello")
    public String hello(@RequestParam String name) {
        ServiceInstance serviceInstance = loadBalancerClient.choose("nacos-provider");
        String url = "http://" + serviceInstance.getHost() + ":" + serviceInstance.getPort() + "/provider/hello?name=" + name;
        return restTemplate.getForObject(url, String.class);
    }
}

2. 配置中心 Demo ​

配置文件 ​

在 Nacos 配置中心创建配置：

  • Data ID: example.properties
  • Group: DEFAULT_GROUP
  • 配置内容:
example.name=Nacos Example
example.description=This is a demo for Nacos
example.version=1.0.0

3

应用代码 ​

java
@Component
@RefreshScope  // 支持配置热更新
public class ExampleConfig {
    
    @Value("${example.name:Nacos Default Name}")
    private String name;
    
    @Value("${example.description:Nacos Default Description}")
    private String description;
    
    @Value("${example.version:1.0.0}")
    private String version;
    
    // getter 方法
    public String getName() { return name; }
    public String getDescription() { return description; }
    public String getVersion() { return version; }
}

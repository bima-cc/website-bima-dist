智能中间件集成验证脚手架 ​

项目简介 ​

智能中间件集成验证脚手架是面向开发者打造的一站式微服务项目脚手架，深度融合 AI 能力与主流中间件集成方案，支持开发者快速完成各类常用中间件服务的集成、验证与测试，大幅提升开发效率。 本项目提供标准化中间件集成方案、统一 Web 界面及配套测试工具，同时集成智能辅助能力，支持 Ollama 本地模型与 OpenAI API 调用。开发者可借助该脚手架快速搭建测试环境、验证中间件功能，并通过 AI 能力加速开发流程，有效缩短项目研发与调试周期。

版本选择 ​

根据您的技术栈和需求，选择适合的智能中间件集成验证脚手架版本：

Spring Boot 版本 ​

  • JDK 版本：17+
  • 主要特性：基于 Spring Boot 3.5.12，提供完整的中间件集成与 AI 能力，适合快速开发和测试
  • 文档链接：Spring Boot 版本

Dubbo 版本 ​

  • JDK 版本：17+
  • 主要特性：基于 Dubbo 微服务框架，适合分布式服务架构场景
  • 文档链接：Dubbo 版本

Spring Cloud 版本 ​

  • JDK 版本：17+
  • 主要特性：基于 Spring Cloud 生态，适合云原生微服务架构场景
  • 文档链接：Spring Cloud 版本

核心功能 ​

所有版本的智能中间件集成验证脚手架都包含以下核心功能：

  • AI接口能力：可调用 Ollama 本地模型和 OpenAI API，支持普通聊天、流式（SSE）聊天、文章写作（SSE）等多种 AI 交互方式
  • 多中间件集成：集成了 Elasticsearch、Kafka、MongoDB、MyBatis、RabbitMQ、Redis、RocketMQ 等常用中间件
  • 统一Web界面：基于 LayUI 框架的直观 Web 界面，方便开发者进行中间件功能测试和验证
  • 多数据源支持：内置多数据源配置，支持同时连接多个数据库
  • OAuth2.0客户端：集成 OAuth2 客户端认证机制，支持以 OAuth2.0 登录统一认证中心（CAS）
  • 完整测试用例：提供了各中间件的测试用例，确保功能正常
  • 多环境配置：支持开发、测试、生产多环境配置
  • 模块化设计：采用模块化设计，便于扩展和定制
  • Docker支持：提供完整的 Docker 部署方案

快速开始 ​

  1. 选择版本：根据您的技术栈和需求选择合适的智能中间件集成验证脚手架版本
  2. 查看文档：点击上方对应版本的链接查看详细文档
  3. 环境准备：按照文档要求准备 JDK、数据库等环境
  4. 配置修改：根据您的环境修改中间件连接等配置
  5. 构建运行：使用 Maven 构建并运行项目
  6. 访问服务：通过 http://localhost:8080 访问脚手架服务

注意事项 ​

  • 环境要求：确保所有依赖的中间件服务都已正确安装和配置
  • 网络要求：构建过程需要访问 Maven 中央仓库，确保网络连接正常
  • AI服务：使用 AI 功能时需要配置相应的模型服务（Ollama 或 OpenAI API）
  • 生产环境：在生产环境中使用前，请进行充分的测试和验证

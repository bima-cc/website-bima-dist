CAS 单点登录系统 ​

CAS (Central Authentication Service) 是 Apereo 基金会维护的企业级开源单点登录系统，为企业应用提供统一的身份认证与授权机制。我们基于不同版本的 CAS Overlay 进行了定制化开发，提供了完整的开发基座、配置模板和部署方案。

主要特性：

  • 架构转型：从传统 WAR 包架构升级为现代 JAR 包架构
  • 自定义认证：实现灵活的身份验证逻辑
  • 安全加固：关闭 CAS 默认账号密码登录，提升系统安全性
  • 数据库集成：集成 MyBatis 数据库访问框架
  • 资源提取：全量提取前端静态资源和模板，方便定制
  • 多语言支持：内置中文、英文等多语言支持
  • OAuth 2.0 支持：完整实现标准 OAuth 2.0 认证流程

版本选择：

  • CAS Overlay 文档 - 包含 7.3.x、6.6.x 和 5.3.x 系列的详细文档

智能中间件集成验证脚手架 ​

智能中间件集成验证脚手架是面向开发者打造的一站式微服务项目脚手架，深度融合 AI 能力与主流中间件集成方案，支持开发者快速完成各类常用中间件服务的集成、验证与测试，大幅提升开发效率。

主要特性：

  • AI接口能力：可调用Ollama本地模型和OpenAI API
  • 多中间件集成：集成了Elasticsearch、Kafka、MongoDB、MyBatis、RabbitMQ、Redis、RocketMQ等常用中间件
  • 统一Web界面：基于LayUI框架的直观Web界面
  • 多数据源支持：内置多数据源配置
  • OAuth2.0客户端：集成OAuth2客户端认证机制

版本选择：

  • 智能中间件集成验证脚手架文档 - 包含 Spring Boot、Dubbo 和 Spring Cloud 系列的详细文档
